Lógica de programação: é a técnica de encadear pensamentos para atingir determinado objetivo. Estes pensamentos, podem ser descritos como uma seqüência de instruções, que devem ser seguidas para se cumprir uma determinada tarefa. 
Seqüência Lógica são passos executados até atingir um objetivo ou solução de um problema.

Metacognição :Habilidade de concentração nos aspectos.

Abstração : O mesmo que Metacognição, só que se concentra nos aspectos essenciais, ignorando caracteres menos importantes.

Algoritmos: Uma sequência de passos que resolveram o problema.

Pseudocódigos: Uma forma genérica de escrever o algoritmo, usando uma linguagem.
